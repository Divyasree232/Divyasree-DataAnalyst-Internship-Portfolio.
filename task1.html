Objective

Rapidly understand the dataset and complete the crucial first stage of analytics: acquire, clean, and prepare data for analysis.

Key Steps

  • Data access and familiarization with the dataset.
  • Create a data dictionary documenting variables, types, and business relevance.
  • Data quality assessment: missing values, duplicates, inconsistent formatting, and outliers.
  • Data cleaning and transformation, with reproducible scripts.
  • Create an analysis-ready dataset for later tasks.

Deliverables

  • Data dictionary
  • Cleaning scripts in Python/Pandas or SQL
  • Cleaned dataset ready for analysis

Timeline: 14 days
